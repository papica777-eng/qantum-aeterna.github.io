🌡️ Thermal-Aware Pool

ThermalAwarePool - CPU temperature management

Automatically adjusts parallelism based on CPU temperature to prevent throttling.

State Temperature Max Instances
🟢 COOL <60°C 40
🟢 WARM 60-70°C 30
🟠 HOT 70-80°C 20
🔴 CRITICAL >80°C 10 
import { ThermalAwarePool } from './src/enterprise/thermal-aware-pool';

const pool = new ThermalAwarePool({
    maxInstances: 40,
    coolThreshold: 60,
    warmThreshold: 70,
    hotThreshold: 80,
    criticalThreshold: 90
});

await pool.start();
// Pool auto-adjusts based on CPU temperature

DockerManager - Selenium Grid orchestration

const docker = new DockerManager({
    hubPort: 4444,

🎖️ Swarm Commander

SwarmCommander - Commander-Soldier pattern

Commander distributes tasks to soldiers (browser instances) for parallel execution. 

import { SwarmCommander } from './src/enterprise/swarm-commander';

const commander = new SwarmCommander({
    minSoldiers: 4,
    maxSoldiers: 40,
    taskTimeout: 30000
});

await commander.initialize();
await commander.queueTask({
    id: 'test-001',
    type: 'browser-test',
    payload: { url: 'https://example.com' }
});

🔊 Bulgarian TTS

BulgarianTTS - Native text-to-speech

🇧🇬 Български текст-към-реч за гласова обратна връзка по време на тестове.

Template Bulgarian Text
test_passed "Тестът премина успешно"
test_failed "Тестът се провали"
error_found "Открих грешка в {element}"
healing "Намерих нов селектор" 
import { BulgarianTTS } from './src/enterprise/bulgarian-tts';

const tts = new BulgarianTTS({ engine: 'auto' });
await tts.initialize();

await tts.speak('Тестът премина успешно');
await tts.speakTemplate('error_found', { element: 'бутон' });
